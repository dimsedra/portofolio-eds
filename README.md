## 🚀 Local Installation & Run Guide

### 📋 Prerequisites
Make sure you have **Node.js** (LTS version recommended) installed.

### ⚙️ Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/dimsedra/portofolio-eds.git
   cd portofolio-eds
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Duplicate `.env.example` as `.env.local` and add your credentials:
   ```bash
   # On macOS/Linux:
   cp .env.example .env.local

   # On Windows:
   copy .env.example .env.local
   ```
   Open `.env.local` and configure your API Keys:
   ```env
   GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
   APP_URL="http://localhost:3000"
   ```

4. **Run the Development Server**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to **[http://localhost:3000](http://localhost:3000)**.

5. **Production Build**
   To build and run the optimized production package:
   ```bash
   npm run build
   npm run start
   ```
